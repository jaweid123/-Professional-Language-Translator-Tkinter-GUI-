# 🌍 Professional Language Translator (Tkinter GUI)

A modern and professional desktop language translator application built with Python and Tkinter.
This application allows users to translate text into multiple languages using Google Translate API.

---

## 📌 Features

* Clean and modern GUI using Tkinter & ttk
* Multiple language support (English, Persian, Arabic, French, German, Turkish, Urdu)
* User-friendly interface
* Error handling system
* Clear button functionality
* Object-Oriented Programming (OOP) structure
* Easy to extend and customize

---

## 🛠 Technologies Used

* Python 3
* Tkinter (GUI)
* ttk Styling
* googletrans API

---

## 📷 Application Preview

Main Components:

* Input Text Box
* Language Selection Combobox
* Translate Button
* Clear Button
* Output Text Box

## 🚀 Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/translator-app.git
cd translator-app
```

### 2️⃣ Install Required Library

```bash
pip install googletrans==4.0.0-rc1
```

### 3️⃣ Run the Application

```bash
python translator.py
```
